Hola soy Marcelo Andreola estudiante de la Tecnicatura en programacion web en la Universidad del Comahue

Experiencia: Tengo experencia en html, css, boostrap en lo que es front-end con algunos proyectos personales como el de una pagina web para un gimnasio donde use boostrap y algunas cosas de css para poder hacer mas responsiva la pagina como los media queries. Pero la parte que mas me interesa es el back-end que tengo experiencia con mysql, phpmyadmin y con la parte de logica con java poo y python poo con algunos proyectos personales como con java poo en netbeans y phpmyadmin un programa de veterinaria donde tenia que registrar al dueño, el tipo de animal, fecha de ingreso, precio de la atencion y con python un programa de una libreria donde se registra un usuario con nombre, apellido, contraseña y email y por otro lado que se puedan subir los libros con su nombre, autor, fecha y si estan disponibles y al final entre esos dos programas poder prestarle los libros disponibles al usuario en la cual use python poo, mysql y una libreria de python que se llama tkinter. Ademas de todo esto uso github y git bash con comandos para poder subir estos programas a un respositorio
